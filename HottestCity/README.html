simpleWeather - jQuery Plugin

A simple jQuery plugin to display current weather data for any location and doesn’t get in your way. Now supports HTML5 GeoLocation! Handcrafted with ♥ from Austin, Texas by James Fleeting. You can find demos and docs at simpleweatherjs.com.
